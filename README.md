# WolfAndSheep


<h1>狼與羊遊戲  
  
<h2>遊戲規則  
<h3>。每回合羊會自動找草吃，狼則會找羊吃，狗則負責趕跑狼  
<h3>。當羊吃了三次草後，玩家會獲得金幣  
<h3>。每回合會生成草與小羊  
<h3>。每6~10回合會出現狼  
  
<h2>此專案主要運用:  
<h3>。Astar尋路系統  
<h3>。網格系統  
<h3>。物件池系統
