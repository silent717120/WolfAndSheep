# WolfAndSheep


<h1>狼與羊遊戲

<h2>遊戲規則  
<h4>。每回合羊會自動找草吃，狼則會找羊吃，狗則負責趕跑狼  
<h4>。當羊吃了三次草後，玩家會獲得金幣  
<h4>。每回合會生成草與小羊  
<h4>。每6~10回合會出現狼  
  
    
<h2>此專案主要運用:  
<h4>。Astar尋路系統  
<h4>。網格系統  
<h4>。物件池系統
